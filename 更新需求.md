# 首页入场与爆炸动画效果

## 背景
页面是纯黑背景，标题「结结恩的四年 / 2022—2026」居中置顶，固定在画面正上方，`pointer-events: none` 不参与交互。

右侧悬浮按钮也需要改进，改成像幽灵一样的玄幻神秘风格，文字是“打开我的记忆”。

与此同时，标题文字用 CSS `transition: opacity 0.6s ease` 同步淡出。爆炸动画结束（约 800ms）后，整个首页 `div` 用 `opacity: 0; pointer-events: none` 隐藏，主页面 `display: flex` 淡入。

## 动画

点击右侧悬浮按钮后，触发爆炸动画：记录每个贴片当前的屏幕坐标作为起始点 `(ox, oy)`，计算该点相对球心的方向向量 `(dx, dy)`，归一化后加上随机扰动作为飞散方向 `(vx, vy)`。之后每帧用 `explodePhase`（从 0 递增到 1）驱动位移：`x = ox + vx * explodePhase * 400`，同时 `globalAlpha = 1 - explodePhase`，实现贴片向四周飞散同时淡出的效果。


## 照片球

1. **卡片不是紧贴球面的平面贴图**，而是每张卡片各自独立，带有明显的**透视倾斜**——靠近球体左右边缘的卡片向内倾斜，靠近顶部的向下倾斜，每张卡片的法线始终朝向球心，像真正钉在球面上一样。

2. **卡片之间有明显间隙**，不是无缝拼接，球面底色（深黑色）从缝隙中透出来，形成黑色网格感。

3. **卡片有圆角**（约 6–8px），有轻微白色高光边框，使每张卡片看起来像实体卡片而不是贴图。

4. **球体有强烈的球形立体感**：靠近边缘的卡片尺寸明显缩小（透视缩放），越往边缘越小、越向内倾斜，而且边缘卡片亮度/饱和度略低，中心区域最亮最清晰。

5. **背面不可见**：z 轴朝向相机反方向的卡片完全不渲染，不是半透明，是直接隐藏，所以球是实心视觉效果。

6. **卡片密度极高**，约 120–200 张卡片铺满整个球面，赤道附近密度最高（因为赤道周长最长），极点附近卡片少且尺寸小。

7. **自转方向**：绕 Y 轴顺时针缓慢自转，速度约 5–8 秒转一圈，匀速无缓动。

8. **背景**：球体正下方有隐约的暗色地面倒影/阴影。

用：Three.js，每张卡片是一个 `PlaneGeometry`，`position` 设置在球面坐标上，`rotation` 用 `lookAt` 朝向球心反方向（即法线朝外）。用 `MeshBasicMaterial` 或 `MeshStandardMaterial` + `map` 贴图。不要用 CSS3D，不要用 Canvas 2D 手动投影。**
